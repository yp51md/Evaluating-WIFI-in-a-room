## 背景
这是计算物理课程论文的实现代码，和zjj、ls合作完成。

## 使用

代码提供三种展示方式。
- 运行`drawLights.py`，可以得到光线的分布图。
- 运行`evaluate.py`，可以得到WiFi的总分布强度的示意图。
- 运行`evaluate2.py`，可以得到保证三点信号最好的情况下，信号的应该放置位置的分析。
